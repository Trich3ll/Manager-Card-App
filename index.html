Embedded / IoT
ESP32 CYD Plane Radar

Ported an open-source ADS-B aircraft radar from an incompatible ESP32-S3 platform to the ESP32-2432S028R "Cheap Yellow Display" — a $15 hobbyist board with a 2.8" ILI9341 touchscreen. Displays live aircraft callsigns, altitudes, headings, and a rotating sweep line in real time. Key challenges: ILI9341 offset_rotation quirks, RGB byte-order swap (orange tint fix), screen flicker eliminated via sprite double-buffering, and RAM exhaustion solved by reducing sprite colour depth 16-bit → 8-bit (115KB → 57KB) to survive WiFi stack allocation. FreeRTOS dual-core architecture keeps HTTP fetches on Core 0 and display loop on Core 1.
